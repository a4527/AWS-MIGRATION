# 08. Portfolio

## 프로젝트 소개

기업 파일 공유 시스템을 온프레미스에서 AWS 클라우드 네이티브 환경으로 전환한 사례를 정리한다.

## 강조 포인트

- Docker Compose 기반 빠른 초기 구축
- Spring Security와 JWT 기반 인증
- MinIO에서 S3로의 저장소 전환
- PostgreSQL에서 Aurora로의 마이그레이션
- ECS Fargate, Terraform, GitHub Actions 기반 운영 자동화

## 결과물

- 아키텍처 다이어그램
- ERD
- API 문서
- 마이그레이션 기록
- 운영/배포 문서
- Terraform 기반 AWS 인프라 설계 정리
